Como Executar o Projeto

Backend
Acesse a pasta do backend pelo terminal:
cd backend-supermercado
Instale as dependências:
npm install
Inicie o servidor:
node Server.js

Frontend

Abra um novo terminal e acesse a pasta do frontend:

cd frontend-supermercado

Instale as dependências:

npm install

Inicie a aplicação:

npm run dev

Após isso, acesse o endereço exibido no terminal.

Observação:

Ao iniciar o sistema pela primeira vez, o backend cria automaticamente um usuário administrador padrão para acesso inicial ao sistema.
